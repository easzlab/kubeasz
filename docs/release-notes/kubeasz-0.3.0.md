## kubeasz-0.3.0 发布说明

- 组件更新：
  - k8s: v1.11.2/v1.10.6/v1.9.10/v1.8.15

  - kube-router: v0.2.0-beta.9

- 其他：
  - 更新 jenkins and plugins (#258)
  - 重写 nfs动态存储脚本与文档
  - 优化 cluster-addon 安装脚本
  - 增加 docker 配置文件

  - fix 多主集群下change_k8s_network时vip丢失问题
  - fix 禁止节点使用系统swap

  - fix Ubuntu 18.04无法安装haproxy、keepalived问题
